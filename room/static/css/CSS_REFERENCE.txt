<!--
1. common.css contains shared styles like:
   - Root color variables
   - Base body/html styles
   - Navigation bar (.navbar-custom)
   - Footer (.footer)
   - Form elements (.form-control, .input-group, .form-check-wrap)
   - Buttons (.btn-primary, .btn-secondary, etc.)
   - Tables (.table, .table-responsive)
   - Status badges (.status-badge, .available, .not-available, .pending)

2. Each template has its own CSS file with styles specific to:
   - Container and card styles
   - Typography (title, subtitle, labels)
   - Grid layouts (dashboard-grid, monument-grid)
   - Card styles (action-card, place-card)
   - Images and galleries
   - Responsive breakpoints

3. To modify styles for a specific page:
   - Open the corresponding CSS file (e.g., add_room.css)
   - Find the relevant selector/class
   - Modify the CSS properties
   - No need to search through the entire theme.css file!

4. If you want to change a style used on multiple pages:
   - Check if it's in common.css (shared across all pages)
   - Or modify it in each individual template's CSS file

5. Color variables are defined in common.css under :root:
   --ink: #172033 (main text color)
   --muted: #667085 (secondary text)
   --line: #d9e1ec (borders)
   --soft: #f5f7fb (light backgrounds)
